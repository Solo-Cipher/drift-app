# DRIFT — Smart Travel Itinerary App

A beautiful travel planning app built with Flutter Web. Create detailed day-by-day itineraries for couples traveling to destinations worldwide.

## Features

- 🗺️ **Day-by-day itineraries** — curated activities, transport, and costs for each day
- ✈️ **Multi-destination** — support for 9 countries with unique content per destination
- 💰 **Cost tracking** — accommodation, transport, and food costs per day
- 🔔 **Smart alerts** — closure warnings, price alerts, weather notifications
- 📱 **Responsive design** — works on desktop and mobile

## Getting Started

```bash
flutter pub get
flutter run -d chrome
```

## Tech Stack

- **Flutter Web** (Dart)
- **GitHub Pages** for deployment
